# Commercial support & pilots

**The AudioTrace library is free and MIT-licensed** ([LICENSE](LICENSE)) — you
never need a commercial license to use, self-host, or embed it.

What we sell is a **managed outcome on top of it**: we set up and run the
regression gate + quality dashboard for your voice agent, so a bad
prompt/model/voice change turns your build red the same way a failing test does
— without you standing up the pipeline yourself.

## The pilot — "early access + we run it for you"

For teams shipping voice agents on Vapi, Retell, LiveKit, Pipecat, Twilio, or a
custom stack:

- **We wire it into your CI.** Your golden calls become a committed baseline;
  every change is gated against it (quality, sentiment, latency, drop-off,
  compliance) via the [AudioTrace GitHub Action](action.yml).
- **You get readable reports.** Per-call and run-over-run deltas a human can
  actually read — what moved, by how much, and whether it regressed.
- **We keep it green.** We tune tolerances, add fixtures, and react to drift
  with you during the pilot. Concierge, not a login-and-figure-it-out SaaS.
